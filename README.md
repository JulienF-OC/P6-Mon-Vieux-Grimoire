# Mon Vieux Grimoire

Projet 6 de la formation Développeur Web OpenClassrooms - API backend pour un site de notation de livres.

## Installation

### Backend
```bash
cd Backend
npm install
```

Créer un fichier `.env` à la racine du dossier Backend avec :
```
MONGODB_URI=votre_uri_mongodb
JWT_SECRET=votre_secret_jwt
```

### Frontend
```bash
cd Frontend
npm install
```

## Lancement

### Backend
```bash
cd Backend
npm start
```

Le serveur démarre sur `http://localhost:4000`

### Frontend
```bash
cd Frontend
npm start
```

L'application s'ouvre sur `http://localhost:3000`
